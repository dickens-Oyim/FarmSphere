# 🌾 FarmSphere

FarmSphere is a full-stack web application designed to connect **farmers, buyers, and administrators** through a unified digital platform.

The project was built using **React + Vite** with a responsive interface inspired directly from the Figma design prototype. It provides role-based experiences for different users, including farmers managing crops, buyers purchasing products, and administrators managing the platform.

---

##  Features

### Authentication System

* Demo login accounts for:

  * Admin
  * Farmer
  * Buyer

###  Role-Based Dashboards

####  Admin Dashboard
* Manage users
* Manage products
* View platform analytics

####  Farmer Dashboard

* Manage crops
* Track sales


####  Buyer Dashboard

* Browse available products
* Track orders

###  Responsive UI

* Clean and modern interface
* Matches the Figma design prototype
* Mobile-friendly layouts using Flexbox and Grid

#  Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/your-username/farmsphere.git
```

### 2. Navigate Into Project

```bash
cd vite-project
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start Development Server

```bash
npm run dev
```

### 5. Open Application

Visit:

http://localhost:5173
```
